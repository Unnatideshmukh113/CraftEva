# 📸 Assets

This folder contains visual assets for the CraftEva project.

## Contents

| File/Folder | Description |
|---|---|
| `logo.png` | CraftEva application logo |
| `screenshots/` | UI screenshots for documentation |

## Usage

Reference assets in the README using:
```markdown

```
